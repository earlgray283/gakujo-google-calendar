# gakujo-google-calendar

## For Developer

### Build

各プラットフォームごとの build から zip は次のコマンドで可能です。

```console
$ make all
```

#### Windows

**Dependencies**

- mingw-w64(https://www.mingw-w64.org/)

#### Linux

**Dependencies**

- Docker(https://docs.docker.com/engine/install/)

#### macOS

**Dependencies**

- clang

```console
$ sudo apt update && sudo apt install -y clang
```
